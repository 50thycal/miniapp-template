# Public Assets

This directory contains static assets for your Farcaster Mini App.

## Required Files

### embed-preview.png
- Image shown when your mini app is shared or embedded
- Recommended size: 1200x630 pixels
- Referenced in `app/layout.tsx` metadata

## Optional Files

You can add additional static assets here:
- Icons (favicon.ico, icon.png)
- Images used in your mini app
- Other static files

## Example Structure

```
public/
├── embed-preview.png    # Main preview image for sharing
├── icon.png             # App icon
├── favicon.ico          # Browser favicon
└── .well-known/         # Farcaster domain manifest directory
    └── farcaster.json   # Auto-generated by Farcaster client
```

## Important

Make sure to update the `domain` field in `app/config.ts` to match your deployed URL so that these assets are properly referenced in your mini app metadata.
